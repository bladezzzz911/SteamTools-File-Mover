SFM | Steam Files Mover
SFM — это легкая и быстрая утилита, разработанная на C# (.NET 10), предназначенная для автоматизации рутинного процесса распределения игровых файлов из архивов по рабочим директориям Steam.

🚀 Основные функции
Умное распределение: Программа автоматически распознает типы файлов и перемещает их в нужные подпапки:

.manifest ➔ config/depotcache

.lua / .st ➔ config/stplug-in

.bin ➔ config/StatsExport

Интуитивный интерфейс: Минимум действий — выберите архивы и укажите путь к Steam, остальное программа сделает сама.

Система логирования: Все операции отображаются в реальном времени с цветовой индикацией статуса.

Кастомизация: Поддержка темной и светлой тем оформления с сохранением настроек.

🛠 Инструкция по использованию
Скачайте последний релиз в разделе Releases.

Запустите файл SFM.exe.

Нажмите кнопку "ВЫБРАТЬ АРХИВЫ И ПУТЬ".

Выберите один или несколько ZIP-архивов с файлами.

Выберите корневую папку вашего Steam (например, D:\Steam).

Дождитесь уведомления "ГОТОВО" в статус-баре.
